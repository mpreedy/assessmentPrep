CSS3

Some notes on things to remember for CSS3 that are a little different than what was originally covered in CSS.

Border Radius

This essentially is rounded corners on any element. This would include a solid border that encloses an element

There should be some sort of text or image and not an empty box.

To get the border, in CSS select the element and type: `border-style:` and how you would like it. This is a `solid` border. To get rounded corners the element should be styled as `border-radius` and how much it should be. This is 15px. The higher the number, the more round it will be.

This is the highest you can go.

Box Shadow

This element has 4 parts: x-val, y-val, blur and black.

This exact element looks like this: `box-shadow: 10px -2px 10px #000;`

Progressive Enhancement

The term 'Progressive Enhancement' refers to the use of newer features that add to the experience in modern browsers that support those features, but doesn't deract from the experience in older browsers.

Essentially it means that if older browsers don't support it then some of the smaller details like box-shadow will not be seen, but it shouldn't detract from your overall layout.
