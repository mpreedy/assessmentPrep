Commandments

While the pun is funny there are some things that I forget about the command line. This is a study reminder about some of those things.

To copy a file

  • cp (filename) (whereto)

    • This is to strait copy a file. Write the file name and the place that it's going

  • cp * (directory)

    • This is to copy all files in a directory

  • cp m*.txt (directory)

    • This copies all 'm' text files

To remove files

  • rm (filename)

    • Removes files

  • rm -r (directory)

    • Removes entire directories. 'r' stands for 'recursive'.

Echoing

  • echo (text)
  • echo (text) > (filename).txt
    • cat (filename).txt

      • This will print the text you entered for the (filename).txt

  • cat (filename).txt > (filename).txt

    • redirects files into the filename on the right

  • cat (filename).txt >> (filename).txt

    combines

Other items

  • | = pipe ex: | wc prints the number of lines, words and characters
  • grep -i (serachterm) (file).txt

    • searches and is case sensitive

  • ~ /.bash_profile

    • This opens an editor that controls the environment of the terminal. This also includes changing what some of the commands can be. For example `pwd` can be changed to `pd` in the terminal.
